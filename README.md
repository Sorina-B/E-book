# E-book

## Project Title
Little Women E-Book App
## Description
A lightweight and elegant Android E-book application built with **Kotlin**. This app provides a dedicated reading experience for the classic novel "Little Women" by Louisa May Alcott. It features a clean, edge-to-edge UI, a synopsis screen, and an easy-to-use chapter navigation system.

## Features
- **Home Dashboard:** Displays the book cover, title, author, and a brief synopsis of the novel.
- **Seamless Navigation:** Uses Android Fragments to navigate smoothly between the home screen and the reading interface.
- **Chapter Selection:** A horizontal scrollable menu allows users to easily jump between Chapters 1 through 4.
- **Scrollable Reading View:** Chapters are loaded dynamically and displayed in a full-screen, scrollable text view for a comfortable reading experience.
- **Local Asset Loading:** Text content is efficiently read and rendered directly from local `.txt` asset files.

## Technologies Used
- Kotlin
